# Costume Design Portfolio

Live Demo: juliamachado.vercel.app

This project is a personal portfolio showcasing custom costume design work, including original concepts and completed pieces.

The goal of this portfolio is to highlight creativity, craftsmanship, and attention to detail through a clean and visually focused presentation.

## Features

- Showcase of costume design projects
- Visual-focused layout for highlighting details
- Responsive design for different screen sizes
- Simple and intuitive navigation

## Tech Stack

- React
- Vite
- CSS / Tailwind

## Getting Started

To run the project locally:

```bash
npm install
npm run dev
